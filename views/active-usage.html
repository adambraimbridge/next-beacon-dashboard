Active usage of feature

How this is calculated:

How many users visited next.ft in a one-week period, two weeks ago? users
Of those users, how many visited next.ft in the last seven days? active users
How many of those active users used this feature?* active, feature-clicking users
What's that as a percentage? %

* These users clicked at least one "" trackable element in the last two weeks.

Active usage for different device sizes

How many users visited next.ft in a one-week period, two weeks ago, on a specific device size? (Large, medium or small)
