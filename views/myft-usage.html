Definitions
    A Next user here is defined as a user with a UUID (i.e. subscribers/registered users that we can recognize).
    A Follow user here is defined as a Next user who, at any point during the period specified, was following at least one topic.
    A MyFT user is classified as anybody who interacted with an article from a myFT source (e.g. email, news feed, tray etc) during the specified period.
