A/B - Performance

Hypothesis: Users engage more if webpages load faster.

Group A: Webpages are delayed by 5 seconds.
Group B: Webpages are not delayed.
  • Since 30th September 2015, all users of next.ft.com have been evenly split into group A and group B.
  • Each time a next.ft.com webpage loads, it checks which group the user is in.
  • If they're in group A, the page is artifically delayed.
  • Group A users don't know they're delayed. Will they notice? Will they opt-out more?
  • Users in group B remain unaffected. Do they have higher RFV scores than group A?

Frequency

Is the average frequency score for each group going up or down? To calulate an average user frequency for a given day:

  1. Count how many users visited any webpage (at least once) that day;
  2. Calculate each user's frequency; (The number of days on which they visited in the 90 days previous to that day.)
  3. Add up all the frequencies and divide by the number of users.
